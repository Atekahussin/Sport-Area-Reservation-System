## 📌 Project Overview
This project is a dedicated management tool designed for sports clubs to streamline customer registration and facility booking. It aims to save time and effort by providing an organized digital alternative to manual record-keeping.

## 🚀 Program Features
The system provides a comprehensive set of functionalities (13 main steps) including:
- **Customer Management:** Add, remove, search, list, and update customer profiles (Name, ID, Phone).
- **Sports Area Management:** Add new sports areas, modify area details (Name, Price, Capacity), and delete existing areas.
- **Reservation Logic:** Link customers to specific sports areas by ID.
- **Search Capabilities:** Fast lookup for customers or sports areas to check existence or status.

## 🛠️ Tech Stack
- **Language:** C++
- **Environment:** Standard C++ Compiler.
- **Concepts:** Loops, Conditional Statements, Arrays/Matrices, and Modular Programming.
